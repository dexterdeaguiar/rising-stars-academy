## ❓ FAQ

**Q: Can we edit the website ourselves?**  
A: Not easily without coding skills. Dexter or a web developer will handle edits.

**Q: Can we move to WordPress or Wix instead?**  
A: Possibly, but this HTML version is lighter, faster, and cheaper to host.

**Q: How much does it cost to keep the website running?**  
A: Netlify is free (or ~$20/month for premium). Domain hosting (~$12/year). Minimal cost.

**Q: Can we add a blog?**  
A: Not with this template. Would need WordPress or a CMS. Talk to Dexter about options.

**Q: How do we handle student applications online?**  
A: Can use Google Forms, Wix Forms, or JotForm (free, embedded on the website).

**Q: How do we know who's visiting the website?**  
A: Setup Google Analytics. Dexter can do this.
